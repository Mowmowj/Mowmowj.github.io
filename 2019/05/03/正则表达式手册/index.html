正则表达式

正则表达式用于定义一些字符串的规则，

计算机可以根据正则表达式，来检查一个字符串是否符合规则，

获取将字符串中符合规则的内容提取出来.

正则表达式语法

方法一: 创建正则表达式 (优点 : 更加灵活)

var 变量 = new RegExp(“正则表达式”,”匹配模式”);

匹配模式

1.忽略大小写 i

2.全局匹配模式 g

e.g.

var r= new RegExp(“a”);//定义规则

方法二: 字面量创建正则表达式 (优点 : 更加方便)

var 变量 = /正则表达式/匹配模式

变量 = /正则表达式/匹配模式

test()方法

符合返回true 不符合返回false

rep = /[ ]/

[ ]

rep = /[ab]/ 表示ab都可以

rep = /[a-z]/表示a-z都行

rep = /[A-z]/ 表示任意字母

rep = /a[bde]c/ 表示a开头 c结尾中间 b|d|e

rep = [.^ab]表示除了ab都行 这里没有小数点\

/^a/ 这里表示开头有a才行

量词

{n} 正好出现n次

{m,n} 出现m-n次

{m,} m次以上

“+” 至少一个，相当于{1,}

“*” 0个或多个，相当于{0,}

“?” 0个或1个，相当于{0,1}

^ 表示开头

$ 表示结尾

e.g.

/x{n}/ x出现n次

/ab{3}/ b出现3次

/(ab){3}/ ab出现3次

/(ab){3}c/ ac中有3个b

/(ab){1,3}c/ ac中有1-3个b

/(ab){3,}c/ ac中有3个以上

/(ab+c/ +号表示 至少一个b

/(ab*c/ 有多少都行 包括0个

/(ab?c/ 有0个或1个相当于{0,1}

/^a/ 这里表示开头有a才行

/a$/ 这里表示结尾有a才行

/^a$/ 这里表示开头结尾都有a才行

e.g.

这里表示手机号的正则表达式

/^1[3-9][0-9]{9}/$

转义

小数点表示检索所有字符

要找 . 要通过转义

/\./

\.才行

\本身也需要转移 \\

正则符号

\w

[A-z0-9_]

任意字母数字下划线_

\W

[^A-z0-9_]

除了字母数字下划线_

\d

[0-9]

表示任意数字

\D

[^[0-9]]

除了数字

\s

空格

e.g.

str.replace(/^\s*/,” “) //去掉开头空格

str.replace(/\s$/,””) //去掉结尾空格

合并

str.replace(/^\s/|/\s\$/g,” “) //去掉开头结尾

str.replace(/^\s+/|/\s+$/g,” “) //一样

\S

除了空格

\b

单词边界

/\b单词\b/

限定边界 独立的单词

\B

除了单词边界
