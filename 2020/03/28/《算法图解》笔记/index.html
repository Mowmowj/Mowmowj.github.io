欧几里得算法

题目：求任意a,b两个正整数的最大公约数？

思路：

比如

1997 / 615 = 3 (余 152)

615 / 152 = 4(余7)

152 / 7 = 21(余5)

7 / 5 = 1 (余2)

5 / 2 = 2 (余1)

2 / 1 = 2 (余0)

至此，最大公约数为1

发现规律是：

上一个的除数作为下一个的被除数，余数作为下一个的除数

那么只需要每次把上一个的除数b赋值给下一个a，

直到最后一步把余数赋值给下一个b ，把除数赋值给下一个a ，发现b=0 退出循环 返回a的值 得到答案

b为0结束循环 返回a
